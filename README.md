# Lancer-TTRPG-Full-Auto

A Foundry VTT v12 module for the Lancer system that adds quality-of-life automation.

## Installation

In Foundry's setup screen:

1. Open **Add-on Modules**.
2. Click **Install Module**.
3. Paste this Manifest URL:

https://github.com/HetzenGN/Lancer-TTRPG-Full-Auto/releases/latest/download/module.json

4. Install the module.
5. Open your Lancer world.
6. Go to **Manage Modules** and enable **Lancer Auto Effects**.

## Compatibility

- Foundry VTT: v12
- Lancer system: v2.x / Foundry v12 line
